@add(globals)
template<typename RE>
std::string random_permute(
std::string str, RE &re
) {
@put(random permute);
return str;
}
@end(globals)
  • The function random_permute randomly shuffles all the characters in a string
  • and return the shuffeled result
  • It receives a generic random engine to ease unit-tests
@def(shuffle pw)
pw = random_permute(pw, re);
@end(shuffle pw)
  • The main function uses random_permute to perturbate the letters from the different categories
@def(random permute)
int i = str.size() - 1;
for (; i > 0; --i) {
Uniform d { 0, i };
std::swap(str[i], str[d(re)]);
}
@end(random permute)
  • random_permute exchanges the last character in the string with a character at a random position in the string
  • Then it exchanges the second to last character with a character at a random position up to and including this string
  • This continues until there is only one character left
@add(unit-tests) {

);
} @end(unit-tests)
  • In this unit-test always the first position is used for swaps
  • So abc becomes cba in the first iteration of the loop
  • Then cba becomes bca and that is the final result
